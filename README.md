# 🚲 Bikcraft

Projeto desenvolvido durante o curso **Bikcraft** da **Origamid**, com foco total em **Front-end**.  
A aplicação consiste em uma **página de venda de bicicletas**, com layout moderno, responsivo e interativo.


## 🧠 Objetivo do Projeto

O objetivo deste projeto foi **praticar e consolidar conceitos fundamentais de Front-end**, incluindo:

- Estruturação semântica com HTML
- Estilização avançada com CSS
- Interatividade e comportamento com JavaScript
- Organização de arquivos
- Boas práticas de desenvolvimento Front-end

---

## 🛠️ Tecnologias Utilizadas

- **HTML5** – estrutura semântica
- **CSS3** – layout, responsividade e animações
- **JavaScript (ES6+)** – interações e comportamentos dinâmicos

> ⚠️ Projeto **100% Front-end**, sem uso de frameworks ou bibliotecas externas.

---

## ✨ Funcionalidades

- Layout responsivo para desktop e mobile
- Navegação intuitiva
- Animações e interações com JavaScript
- Página focada em conversão e apresentação de produtos
- Código organizado e seguindo boas práticas
